# 测试指南

## 构建产物

文件位置: `dist/笔趣阁下载器.user.js`
- 大小: ~3370 行（原 4880 行 → 减少 31%）
- 格式: 单文件油猴脚本
- 包含: 完整油猴头部 + 所有模块

## 模块架构

```
src/
├── main.js                 # 入口文件
├── core/                   # 核心模块 (5 个)
│   ├── config.js           # 配置管理
│   ├── http-client.js      # HTTP 请求
│   ├── site-detector.js    # 站点检测
│   ├── content-cleaner.js  # 内容清洗
│   └── throttle.js         # 智能限流
├── rules/                  # 规则模块 (4 个)
│   ├── rule-analyzer.js    # 规则分析器
│   ├── clean-rule-manager.js    # 清洗规则管理
│   ├── site-rule-manager.js     # 站点规则管理
│   └── element-picker.js   # 元素选择器
├── download/               # 下载模块 (5 个)
│   ├── orchestrator.js     # 下载编排
│   ├── content-fetcher.js  # 内容获取
│   ├── promise-pool.js     # 并发池
│   ├── progress-tracker.js # 进度跟踪
│   └── retry-manager.js    # 重试管理
├── ui/                     # UI 模块 (4 个)
│   ├── theme.css           # 主题样式
│   ├── modal-templates.js  # 弹窗模板
│   ├── modals.js           # 弹窗管理
│   ├── toast.js            # 通知提示
│   └── progress-bar.js     # 进度条
├── quality/                # 质量检测 (1 个)
│   └── content-detector.js # 内容检测
└── data/                   # 数据 (1 个)
    └── site-selectors.js   # 站点选择器
```

## 测试步骤

### 1. 安装到油猴

**方法 A - 直接安装**
```bash
# 在浏览器中打开文件（自动触发油猴安装）
open dist/笔趣阁下载器.user.js
```

**方法 B - 手动复制**
1. 复制 `dist/笔趣阁下载器.user.js` 全部内容
2. 打开 Tampermonkey → 添加新脚本
3. 粘贴代码并保存
